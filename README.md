# ios-blockid-demo-app

1. The demo app uses BlockID SDK as pod file.
2. Xcode - 13.0
3. Demo App version - 1.6.10
4. BlockID SDK version - 1.6.10

NOTE: After installing pods using pod install command in Terminal
1. Select Pods.xcodeproj and open project settings 
2. Select Build Settings
3. Set Build Libraries for Distribution to YES (for both Debug and Release mode)
